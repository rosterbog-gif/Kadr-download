# Конфиденциальность

«Кадр» работает локально на компьютере пользователя.

- приложение не требует регистрации;
- видео, аудио и изображения не отправляются на серверы;
- телеметрия, реклама и аналитические трекеры не используются;
- интернет для монтажа и экспорта не требуется.

При скачивании программы через GitHub действуют правила конфиденциальности
самого GitHub. Это не меняет локальный характер работы приложения.
