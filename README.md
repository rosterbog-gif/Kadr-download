# Кадр

Простой локальный видеоредактор для Windows 10 и Windows 11 от **Rostyan**.

## Скачать

Откройте раздел **Releases** справа на странице и скачайте файл
`Kadr-Setup-1.0.0-win-x64.exe` из последней версии.

После установки ярлык «Кадр» появится на рабочем столе и в меню «Пуск».

## Возможности

- обрезка, разделение, удаление и перестановка видео;
- отдельные дорожки видео, наложений, текста и музыки;
- фото и видео поверх кадра, хромакей и текст;
- перемещение и изменение размера наложений мышкой;
- автоматическое разрешение без чёрных полей;
- экспорт MP4 с H.264 и AAC;
- светлая и тёмная темы;
- полностью локальная работа без регистрации, рекламы и отправки файлов.

## Системные требования

- Windows 10 или Windows 11 x64;
- установка .NET и FFmpeg отдельно не требуется.
