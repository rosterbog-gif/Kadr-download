# Безопасность

Не публикуйте сведения об уязвимости в обычном Issue до её исправления.
Используйте кнопку **Report a vulnerability** в разделе Security репозитория,
если она доступна. В сообщении укажите версию приложения, затронутую функцию и
безопасный способ воспроизведения проблемы.

Не отправляйте настоящие личные видео, пароли, ключи доступа и другие секреты.
